
A project to learn osdev


## FEATURES

* buddy system
* slab
* LinkedList memory allocator
* fat32 ttyfs
* hpet, tsc
* interrupt manager
* RR scheduler
* framebuffer, keyboard
* fork, execve, wait api
* a simple shell


## DEBUG

```shell
make limine
make debug
```

and start debug(F5) in vscode

### RUN

```shell
make limine
make run
```
